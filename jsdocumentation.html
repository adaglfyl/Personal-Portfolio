Intro

JavaScript is a high-level programming language that is dynamically typed. High level languages are generally more intuitive to write, as they are not as closely aligned with computer architecture as low-level languages. Dynamically typed language simplify this process further, as data types of variables are checked during runtime instead of when compiling, removing the need to delcare variable types when writing code.

JavaScript is commonly used for websites and web applications to help with more complex and interactive elements, beyond the static HTML code.

console.log(Hello World!);
Basic Programming Concepts

This documentation assumes that you are familiar with some basic programming concepts, including variables, conditional statements, data types, functions, and object literals.

Variables are a way of storing data to be referenced and manipulated in computer programming and web development. Storing variables means that we can later access or change this data through various means. Each variable will have a data type such as:

  • integer - 9
  • string - 'example'
  • undefined
  • boolean - true / false
  • nothing - null

Variables

Variables act as containers for values of various data types and are named by an 'identifier'. These identifiers must follow set rules:

  • Can contain numbers and alphabetic characters.
  • Cannot be the same as keywords.
  • Cannot contain special characters except for $ and _.
  • First character cannot be a number.
They can be declared in a number of ways in JavaScript ever since the introduction of new declaration syntax in ES6. The keywords used to declare variables in JS are var, let, & const. A variable must be declared before it is used, even if it is undefined.

The var keyword was the original syntax to declare variables prior to ES6. The scope of a variable declared with var is either the global scope or the function scope if declared within a function.

var name = 'Ada';
console.log('My name is' + name + '.');

This will output the following to the console:

My name is Ada.

The let keyword was introduced in ES6 and declares a variable within the block scope. Variables declared with let can be reassigned at any point.

let num = 25;
num = 30
console.log(num);

This will output the following to the console:

30

The const keyword was also introduced in ES6 and also declares a variable within the block scope. The difference is that these variables cannot be reassigned. They are however still mutable.

const pet = 'cat';
pet = 'dog';
console.log(pet);

This will output an error to the console:

Cannot assign to "pet" because it is a constant
Conditional Statements

Conditional statements within JavaScript allow code to run based on set conditions. This is useful for allowing certain blocks of code to run depending on variable values. Some common conditional statements include if, else, and else if.

Example of an if statement:

if (7>3) {
var sentence = "That is correct";
}
console.log(sentence);

Output

"That is correct"

Example of an else statement:

if (3>7) {
let size = "large enough";
} else {
let size = "too small";
}
console.log(size);

Output

"too small"

Example of an else if statement:

if (false) {
let line = "if line"
} else if (true) {
let line = "else if line"
} else {
let line = "else line"
}
console.log(line);

Output

else if line
Functions

Functions are reusable blocks of code that perform a task, being given an input and returning an output. Functions must first be defined before being called.

In JavaScript, a function declaration is peformed by the word 'function', followed by:

  • The name of the function.
  • A list of parameters to the function, enclosed in parentheses and separated by commas.
  • The JavaScript statements that define the function, enclosed in curly braces, { /* â€¦ */ }.
Functions can also be called without being named by simply missing out the name of the function, this is known as an anonymous function, and these cannot be reused.

Example of a function declaration:

function squareNumber(number){
return number * number
};

Once declared, a function may be called by typing the name of the function, followed by the parameters enclosed in parentheses and separated by commas.

Example of a function being called

let myNumber = squareNumber(3);
console.log(myNumber);

Output

9
Objects

Object are a common data structure within JavaScript that contain key-value pairs of data. The data itself is the value while the keys are used to access these values. Keys can be strings, numbers, or symbols. Values can be any data type, including objects.
Objects can be declared by using curly braces, with the key-value pairs either side of a colon and separated by a comma. This is known as an object literal.

Example of an object literal:

const me = {
name: 'Ada',
age: 28,
gender: 'female'
};

To access the values stored in an object, we can use either dot notation or bracket notation, as follows:

const name = me.name; // Ada
const age = me['age']; // 28

To add and delete key-value pairs from an object, we can use dot notation and the delete keyword.

me.country = 'United Kingdom'; // dot notation
delete me.age; // delete keyword to delete key-value pair
